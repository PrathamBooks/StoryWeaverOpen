StoryWeaver ♥ Open-Source

Welcome to StoryWeaver from Pratham Books, a digital repository of multilingual stories for children. StoryWeaver is attempting to address the scarcity of books for children by creating an open and collaborative framework where the creators and consumers of children’s content could interact with one another. Users can read, create, translate, download, and share stories for free. Now, every child can have access to an endless stream of stories in her mother tongue.

StoryWeaver is built on Rails, a web application framework with ReactJS as its front end technology. The third party libraries we use are also open-source. We have released our most useful and active codebase under the MIT license.

We have chosen the OSI-approved MIT license because it makes no restrictions on who uses our codebase or for what purpose. We believe this will enable wide adoption of code base for various purposes even among commerical enterprises

All the code that we release are matured. The platform continues to be actively developed by Pratham Books and new code will be released as soon as they pass internal QA processes.

All our code is on Github where anyone can view and participate in further development of StoryWeaver according to our inclusive participation guidelines. Contributions of any kind, including bug reports, patches, suggestions, and feature requests, are welcome.

There are multiple anticipated beneficiaries of the open source StoryWeaver code base. For Educational organizations it is an excellent starting point to build multilingual collaborative digital books/materials for children. It can be used for building any platform that allows users to read, create, translate, download, and share visual and text content. For developers and freelancers, there are several standalone components and algorithms like recommendation engine, image processing and cropping and others which can be used in other projects. The potential of our code is endless and we are excited to see what you will build with this.

If you are interested in adopting StoryWeaver platform for your needs, our developer team is open for conversations at swopensource.support@prathambooks.org
